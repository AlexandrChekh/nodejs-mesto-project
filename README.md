# Бэкенд Mesto. Каркас API Mesto

## Используемые технологии и решения
- Typescript в качестве основного языка проекта
- Mongodb и ODM Mongoose для хранения данных пользователей
- Node.js в качестве среды выполнения

## Переменные окружения
Секретный ключ JWT не хранится в репозитории. Для локального запуска проект может стартовать без `.env`: в режиме разработки будет использован встроенный `dev`-секрет.

Для production-подобного запуска создайте `.env` по примеру `.env.example` и задайте `JWT_SECRET`.

Перед отправкой работы на проверку убедитесь что она соответствует требованиям проектной работы
